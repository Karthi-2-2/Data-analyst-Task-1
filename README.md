# Netflix Data Cleaning Project

## Objective
The objective of this task is to clean and preprocess a raw Netflix dataset using Python and Pandas.

## Dataset Used
- Netflix Movies and TV Shows Dataset

## Tools Used
- Python
- Pandas
- Google Colab

## Steps Performed
1. Loaded the dataset using Pandas
2. Checked missing values using isnull()
3. Removed duplicate rows
4. Cleaned column names
5. Converted date_added column into datetime format
6. Filled missing values with "unknown"
7. Exported the cleaned dataset

## Output
- Cleaned dataset saved as:
  cleaned_netflix_dataset.csv

## Conclusion
The dataset was successfully cleaned and prepared for further data analysis.
